Quick Start

まずは 3 ステップだけ覚えれば大丈夫です

このツールは、2つの FFLogs 公開ログ URL を読み込み、戦闘とプレイヤーを選んでタイムライン比較を行います。 FFLogs の連携や API 設定は不要です。

比較ページを開く FFLogs を開く
1

公開ログ URL を 2 つ貼る

トップページの ログURL A と ログURL B に、比較したい FFLogs レポート URL を 1 つずつ入力して 読み込み開始 を押します。

非公開ログは取得できません。FF Logs 側で公開されているレポートのみ対応です。

2

戦闘を選んでプレイヤー一覧を取得する

A と B それぞれで比較したい戦闘を選び、この戦闘でプレイヤー一覧を取得 を押します。

ここでは Kill 戦闘のみ一覧に出ます。同じコンテンツ同士を選ぶと比較しやすいです。

3

プレイヤーを選んで比較する

A と B のプレイヤーを 1 人ずつ選び、比較を開始 を押すとタイムライン比較が表示されます。

比較後はタブ切替、フェーズ選択、ズーム、ログ確認が使えます。

つまずきやすい点

  • FFLogs 側で非公開のレポートは読み込めません。
  • プレイヤー一覧は、先に戦闘を選ばないと取得できません。
  • 異なるボス同士は比較できません。同じ encounter を選んでください。
  • 表示がおかしいときは Log / Error Log を確認すると原因を追いやすいです。

準備できたら戻る

手順を確認したら、下のボタンから比較ページに戻ってそのまま操作できます。
