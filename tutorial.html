Quick Start

まず何をすればいいかだけを順に確認

このツールは、2件の FFLogs URL を読み込み、戦闘とプレイヤーを選んでタイムライン比較を行います。 下の 4 ステップを上から順に進めれば、比較表示まで辿り着けます。

比較ページを開く FFLogs を開く
1

FFLogs と連携する

トップページの FFLogs と連携 ボタンを押します。認証画面が開くので、そのまま許可してください。

認証後は自動で比較ページに戻ります。戻ったら右上の状態が「連携済み」になっていることを確認します。

2

比較したい 2 件のログ URL を入れる

URL A と URL B に、それぞれ比較したい FFLogs レポートの URL を貼り付けて 読み込み を押します。

読み込みが成功すると、次の戦闘選択エリアが表示されます。

3

戦闘を選んでプレイヤー一覧を取得する

A と B それぞれで比較したい戦闘を選び、プレイヤー一覧を取得 を押します。

同じボス同士を比較したい場合は、ここで同一コンテンツの戦闘を選んでください。

4

プレイヤーを選んで比較する

A 側と B 側のプレイヤーを 1 人ずつ選択し、比較開始 を押します。

比較結果が開いたら完了です。タブ切替、フェーズ選択、ズーム、ログ確認が使えます。

つまずきやすい点

  • URL の読み込み前に FFLogs 連携が必要です。
  • プレイヤー一覧は、先に戦闘を選ばないと取得できません。
  • 別ボス同士は比較できません。同一 encounter を選んでください。
  • 比較後は下部の Log / Error Log を見ると、進行状況や異常の切り分けがしやすいです。

準備できたら戻る

内容を確認したら、下のボタンから比較ページへ戻ってそのまま操作できます。
